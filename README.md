# Шифратор
Фронтенд-сервис для шифрования

## Доступ к сервису
https://xx-arteeem-xx-learn-is-pt-2-0fbb.twc1.net/

## Для разработчиков

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

## Для DevOps 
Сервис развернут с помощью **Timeweb App Platform**. При коммите в ветку main производится автоматический деплой приложения.
